#  Implemented Data Structures

## 1️. Structures
**Project:** Hospital Patient Record System

Concepts used:
- `struct`
- Array of structures
- Record insertion and deletion

Operations:
- Add patient
- Display patients
- Delete patient record

---

## 2️. Stacks
**Project:** Undo / Redo Text Editor

Concepts used:
- Stack (LIFO principle)
- Two-stack system

Operations:
- Insert operation
- Undo operation
- Redo operation
- Display current text

---

## 3️. Queues
**Project:** Ticket Booking System

Concepts used:
- Queue (FIFO principle)
- Customer service simulation

Operations:
- Add customer
- Serve customer
- Display waiting queue

---

## 4️. Linked Lists
**Project:** To-Do List Manager

Concepts used:
- Singly Linked List
- Dynamic memory allocation

Operations:
- Add task
- Delete task
- Mark task as completed
- Display tasks

---

## 5️. Trees
**Project:** Directory Management System

Concepts used:
- Tree data structure
- Hierarchical relationships

Operations:
- Add file/folder
- Display directory tree
- Search file/folder
- Delete node

This simulates how **operating systems manage file systems.**

---

## 6️. Graphs
**Project:** City Navigation System

Concepts used:
- Graph (Adjacency Matrix)
- Weighted edges
- **Dijkstra’s Algorithm**

Operations:
- Add cities
- Add roads
- Display city map
- Find shortest path

This models **GPS navigation systems like Google Maps.**

---

## 7️. Hashing
**Project:** Employee Database

Concepts used:
- Hash Tables
- Hash Function
- Collision Handling using **Linear Probing**

Operations:
- Insert employee
- Search employee
- Delete employee
- Display all records

This simulates **fast lookup systems used in databases.**

#  How to Run the Programs

1️. Compile the C file

```

```

2️. Run the executable

```
.\program.exe
```

Example:

```
gcc ticket_booking.c -o ticket_booking.exe
.\ticket_booking.exe
```
